# raspberry-pi-fun

Monorepo of Raspberry Pi projects for home automation and experimentation.

## Projects

- **home-assistant-kiosk/** - Chromium kiosk setup for displaying a Home Assistant dashboard on a Raspberry Pi with a connected display.
- **camera-ai/** - (In development) AI-powered camera project for Raspberry Pi.

## Environment

- Target hardware: Raspberry Pi (Linux/ARM)
- Development machine: Windows 10

## Conventions

- Each project lives in its own top-level directory with its own CLAUDE.md for project-specific context.
- Shell scripts target bash on Raspberry Pi OS (Debian-based).
